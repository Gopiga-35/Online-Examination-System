# Gradify – Online-Examination-System

 Gradify – Online Examination System, developed using the MERN Stack.

## Features

Upload and manage exam questions

Student login system

Display of multiple-choice questions (MCQs)

Answer submission and evaluation

Teacher dashboard to manage exam data
